Chosen APIs

Eventful

365 days of the year singles and families, friends and acquaintances, teens and baby boomers rely on Eventful to plan their leisure time and spend their discretionary income. Eventful is the largest digital media company connecting consumers with localized events, entertainment and movies.


We chose Eventful because we needed to gather useful information on events in the user's area in order to present them to the user at all!

Google Places

The Places API is a service that returns information about places using HTTP requests. Places are defined within this API as establishments, geographic locations, or prominent points of interest.


We chose Google places to grab the coordinates of a user's location (or their zip code) into an address to properly tell their distance of the user from different events


Ticketmaster

The Ticketmaster API offers access to live events, attractions, venues and tickets! The datasets span all North American and International markets. Furthermore, the Ticketmaster API provides data coming from ticketing platforms like TicketWeb, Universe and Front Gate.


We chose to use Ticketmaster mainly so we would be able to offer the ability to get tickets to the event the user wished to attend (if the event requires as much)
